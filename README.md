### アプリ名
CompanyPlanner

### アプリについて
ビジネスや組織にとって役立つ、包括的な計画やスケジュール管理機能を提供します。
googleカレンダーのようなUIで、rakumoカレンダーのように会社間で共有できるSPAアプリです。
